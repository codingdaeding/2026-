포트폴리오 멀티 페이지(HTML) 구성

- index.html : 홈
- about.html : 소개
- skills.html : 기술
- projects.html : 프로젝트
- experience.html : 경험
- education.html : 학력/자격
- contact.html : 연락
- assets/style.css : 공통 스타일

사용 방법
1) index.html을 브라우저로 열면 됩니다.
2) 각 페이지의 내용을 본인 정보로 수정하세요.
3) GitHub Pages 등에 올릴 때는 폴더 그대로 업로드하면 됩니다.
